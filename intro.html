Regexes...

  • A way to describe structure of a string
  • ... that looks very messy

Usage

  • Text Editors
  • Command Line Tools
  • Form Validation
  • Route description
  • Wherever need be...

[+-]?(\d+(\.\d+)?|\.\d+)([eE][+-]?\d+)?

Examples

  • a+b  matches   ab, aaaab
  • [0-9]+  matches   0312, 23
  • ([0-9]x)+  matches   3x5x, 0x

Find digits in file

cat example | grep --color=auto -P '[0-9]+'

Reset fontsize in config file

sed "s/^\(fontsize *= *\).*$/\124/" font.config

We find a match for ^\(fontsize *= *\).*$ and replace it by \124

With parentheses you can mark groups and access them later using \1, \2, ...
